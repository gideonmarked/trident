== Description ==

Trident is a custom WordPress theme built specifically for the TRIDENT tumbler e-commerce website. It features a modern, responsive design with integrated product catalog functionality and seamless integration with the Tumbler Seller plugin.

= Key Features =

* Modern, responsive design
* Custom page templates for homepage and product catalog
* Integration with Tumbler Seller plugin
* Tailwind CSS framework
* Custom post types support
* SEO optimized
* Fast loading
* Mobile-friendly
* Customizable through WordPress Customizer

= Page Templates =

* **Homepage Template** - Custom landing page with hero section, product showcase, and call-to-action
* **All Products Template** - Product catalog page with filtering and search capabilities

= Customizer Options =

* Hero section customization
* Color scheme options
* Typography settings
* Layout options

== Installation ==

1. Upload the `trident` folder to the `/wp-content/themes/` directory
2. Activate the theme through the 'Appearance' menu in WordPress
3. Configure theme options in Appearance > Customize
4. Set up your pages using the custom page templates

== Frequently Asked Questions ==

= Does this theme work with the Tumbler Seller plugin? =

Yes! This theme is specifically designed to work seamlessly with the Tumbler Seller plugin. It includes custom page templates that integrate with the plugin's shortcodes and functionality.

= Is this theme mobile-friendly? =

Absolutely! The theme is fully responsive and optimized for all device sizes.

= Can I customize the colors and fonts? =

Yes, you can customize the theme through the WordPress Customizer. Go to Appearance > Customize to access all customization options.

= Does this theme support custom post types? =

Yes, the theme supports custom post types and is specifically optimized for the Tumbler Product custom post type from the Tumbler Seller plugin.
